# What is this?

This is a browser-based CV generator.

# How to use

1. Open [this page](https://akienmain.github.io/cv-generator/)
2. Choose information file (.json)
    - [Example](https://github.com/AkienMain/cv-generator/blob/main/input_example.json)

3. Choose image file (.jpg, .jpeg, .png)
    - [Example](https://github.com/AkienMain/cv-generator/blob/main/face_example.png)

4. Push "Generate CV" button
5. Print it by browser's function (Ctrl + P in Google Chrome)
    - [Example](https://github.com/AkienMain/cv-generator/blob/main/CV_example.pdf)
