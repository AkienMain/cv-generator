# What is this?

This is a browser-based CV generator.

# How to use

1. Choose information file (.json)
    - [Example](https://github.com/AkienMain/cv-generator/blob/main/input_example.json)

2. Choose image file (.jpg, .jpeg, .png)
    - [Example](https://github.com/AkienMain/cv-generator/blob/main/face_example.png)

3. Push "Generate CV" button
4. Print it by browser's function (Ctrl + P in Google Chrome)
